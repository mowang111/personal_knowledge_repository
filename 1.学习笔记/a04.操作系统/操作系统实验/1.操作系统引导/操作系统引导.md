# 任务一 打印提示信息
>bootsect.s能在屏幕上打印一段提示信息“XXX is booting...”，其中XXX是你给自己的操作系统起的名字，例如LZJos、Sunix等（可以上论坛上秀秀谁的OS名字最帅，也可以显示一个特色logo，以表示自己操作系统的与众不同。）

主要修改/boot/bootsect.s中的代码：
```as
msg1:
        ; 插入回车换行符
        .byte 13,10
        ; 定义消息内容
        .ascii "Loading system ..."
        ; 再次插入回车换行符
        .byte 13,10,13,10
```

上述信息共24个字符，下面将他修改位“crzos is booting ...”
