## 特殊场景

为了保证点击输入的正确性，需要保证两个因素——操作对象和操作动作

+ 操作对象
  + 操作对象的正确性
    + 不能操作成了其他元素
  + 操作对象本身属性（可见性，可操作性）
+ 操作动作
  + 发送动作的正确性
  + 动作是否被成功处理
    + 窗口层面
    + 后台层面
      + 比如网页的话，有些自动化接口输入，不会被后台识别



输入错误的特殊场景就需要影响上面的因素中考虑：

  + 操作的元素本身属性（可见性，可操作性）
