# thrift在线匹配系统
## 代码结构
有三个文件夹
```
match_system game thrift
```
match_system为服务端的代码
game为客户端的代码
thrift定义接口

## thrift接口生成
thrift接口
通过thrift接口生成服务端和用户端代码
```bash
thrift -r --gen <language> <Thrift filename>
```
编译thrift生成的代码
```bash
g++ -c main.cpp match_server/*.cpp
```
链接.o文件
```bash
g++ *.o -o main -lthrift
```
